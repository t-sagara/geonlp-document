====================================================
開発者向け資料
====================================================

本書では、GeoNLP の機能を利用して、住所や地名の検索、その経緯度の算出（ジオコーディング）、および自然言語文から地名抽出を行うアプリケーションを開発するユーザを対象に、必要な情報を提供します。

「 :ref:`developers_feature` 」では、GeoNLP と関連サービスの機能を比較し、どのサービスを利用すれば良いのかガイドラインを示します。

「 :ref:`developers_showcase` 」では、GeoNLP の機能を利用して開発したアプリケーションをまとめています。

「 :ref:`documents_index` 」では、主に Web アプリケーションの開発者を対象に、 :ref:`webapi` とデータフォーマットについての情報を提供します。

「 :ref:`software_index`  」では、独自サーバ上にソフトウェアをダウンロードして、 GeoNLP の機能を利用したアプリケーションを開発するパワーユーザ向けの情報を提供します。

「 :ref:`software_license` 」では、 :ref:`webapi` を利用する場合、および GeoNLP ソフトウェアを :ref:`software_download` して利用する場合のライセンスを示します。
