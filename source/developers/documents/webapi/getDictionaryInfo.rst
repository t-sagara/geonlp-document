================================================
getDictionaryInfo
================================================

説明
---------------------------------------

辞書に関する詳細情報を取得する。

文法
---------------------------------------

object **geonlp.getDictionaryInfo** (int[] dictionary_ids)

リクエストパラメータ
---------------------------------------

*dictionary_ids*
 - 情報を取得したい辞書の id の配列
 - 要素数の上限なし

リクエストの例
++++++++++++++++++++++++++++++++++++++++
.. code-block:: javascript

  { 
    "method": "geonlp.getDictionaryInfo",
    "params": [ [28, 29] ],
    "id": "6"
  }


レスポンス
---------------------------------------

リクエストで指定した辞書 id をキー、対応する辞書情報を値とする
object を返す。
