====================================================
開発用リファレンス
====================================================

:ref:`dicapi` では、共用地名辞書サーバから情報を受け取る API について説明します。

:ref:`webapi` では、独自のアプリケーションを構築する場合に
利用可能な :ref:`webapi` のメソッドと、そのリクエスト、レスポンスについて
説明します。

:ref:`datamodel` では、GeoNLP を利用したプログラムを開発する際の
基本的なデータ単位である地名語、地名解析辞書、住所についての
データモデルについて説明します。

:ref:`json` ではこれらのデータが :ref:`webapi` で利用される際に
どのように JSON 表記されるかを示します。
